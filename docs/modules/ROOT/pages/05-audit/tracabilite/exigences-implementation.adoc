= Matrice de traçabilité — exigences ↔ code
:description: Mapping entre les exigences fonctionnelles du PoC et leur implémentation

== Objectif

Permettre à un auditeur ou à un jury de vérifier **en un coup d'œil** qu'une exigence est implémentée, où, et comment la tester.

== Maintenance

Cette matrice **doit être mise à jour** :

* à chaque nouvelle feature (ajouter une ligne avec référence code + test) ;
* à chaque refactor touchant un fichier cité ;
* à chaque release candidate (re-vérification complète).
