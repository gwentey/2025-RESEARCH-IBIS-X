= Plan de tests — matrice
:description: Cas de test par module × niveau

Matrice prescriptive : pour chaque module, quels tests doivent exister (unit / intégration / E2E).

== Légende

* ✅ — cas obligatoire (bloquant si manquant)
* ⚠️ — recommandé
* — — non applicable
