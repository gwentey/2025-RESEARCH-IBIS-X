== À qui s'adresse ce dossier ?

Cette page est le **point d'entrée** pour :

* un *jury académique* (Master 2 MIAGE) qui évalue la rigueur technique, la conformité et la traçabilité du projet ;
* un *auditeur externe* qui vérifie sécurité, RGPD et éthique IA ;
* un *relecteur technique* qui veut comprendre les décisions d'architecture sans lire tout le code.

== Vue synthétique du projet

* *Nature* : Proof of Concept académique (Master 2 MIAGE).
* *Domaine* : plateforme d'expérimentation XAI (Explainable AI) avec pipeline ML.
* *Stack* : 4 microservices Python (FastAPI) + frontend Angular 19, orchestrés par Kubernetes.
* *Données* : datasets publics (Kaggle), utilisateurs de la plateforme (email, mot de passe haché, OAuth Google).
* *Environnements* : Minikube (local dev), AKS Azure (prod).

Voir xref:03-technique/architecture/vue-ensemble.adoc[Vue d'ensemble architecture] pour le schéma C4.
