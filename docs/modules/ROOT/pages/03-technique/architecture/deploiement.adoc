= Topologie de déploiement
:description: Kubernetes — namespace ibis-x, Deployments, Services, Ingress

Tous les composants sont déployés dans le namespace Kubernetes **`ibis-x`**. Gestion via https://kustomize.io[Kustomize] : `k8s/base/` + overlays `k8s/overlays/minikube/` (local) ou `k8s/overlays/azure/` (prod).

== Skaffold — 6 artefacts

Déclarés dans `skaffold.yaml`, construits via Docker multi-stage :

. `ibis-x-frontend` (nginx + Angular build)
. `ibis-x-api-gateway`
. `ibis-x-service-selection`
. `ibis-x-ml-pipeline` (même image pour service web + worker Celery)
. `ibis-x-xai-engine` (service web)
. `ibis-x-xai-engine-worker` (image séparée via `Dockerfile.worker`)

== Configuration

* *ConfigMaps* : URLs internes, flags, paramètres non sensibles, injectés via `envFrom`.
* *Secrets K8s* : `jwt-secrets`, `oauth-secrets`, `kaggle-secrets`, `openai-secret`, credentials BDD/MinIO.
* *En local* : `update-local-secrets.py` synchronise `.env` → `k8s/overlays/minikube/secrets.yaml`.
* *En prod* : Terraform provisionne AKS + secrets via Azure Key Vault.
