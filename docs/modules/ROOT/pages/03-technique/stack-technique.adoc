= Stack technique raisonnée
:description: Choix + justifications

Source de vérité : link:../../../../memory-bank/conventions.md[`memory-bank/conventions.md`]. Cette page en donne la vue synthétique avec justifications (ADRs liés).

== Backend

[cols="1,2,3"]
|===
| Couche | Choix | Justification / ADR
| Langage | Python 3.11 | Écosystème ML (scikit-learn, SHAP, LIME), typage moderne
| Framework web | FastAPI ≥ 0.100 | Async natif, Pydantic, OpenAPI auto, perf
| Validation | Pydantic v2 + `BaseSettings` | Validation + typage + config sans hardcoding
| ORM | SQLAlchemy 2.x | Standard Python
| Migrations | Alembic | Versionnées, indépendantes par service (xref:05-audit/adr/0003-postgres-partage-version-tables.adoc[ADR 0003])
| Auth | fastapi-users | JWT + OAuth clé en main
| Tâches async | Celery + Redis | xref:05-audit/adr/0002-celery-vs-alternatives.adoc[ADR 0002]
|===

[cols="1,2,3"]
|===

== Ce qui **n'est pas** utilisé (et pourquoi)

* **Tailwind CSS** — contredit le choix Angular Material (ADR 0004).
* **Bootstrap** — idem, Material est la base.
* **Docker Compose seul** — incompatible avec l'objectif pédagogique Kubernetes.
* **MongoDB** — relation inter-entités favorise RDBMS.
* **Kafka** — surdimensionné pour le PoC.
* **Dramatiq / RQ** — Celery plus standard (ADR 0002).
* **PyTorch / TensorFlow** — hors scope PoC (sklearn suffit).

== Maintenance

Toute modification majeure de la stack (ajout/suppression d'une lib, changement de version majeure) :

. Nouvel ADR documentant la décision.
. Mise à jour `memory-bank/conventions.md`.
. Mise à jour `CHANGELOG.md`.
