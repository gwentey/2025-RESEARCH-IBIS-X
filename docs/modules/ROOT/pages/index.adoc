= IBIS-X — Documentation
:description: Plateforme XAI & Pipeline ML — Master 2 MIAGE

== Démarrage rapide (10 minutes)

[source,bash]
----
git clone <URL_DU_DEPOT>
cd 2025-research-exai
cp .env.example .env   # Remplir les 8 variables obligatoires

# Windows (Git Bash) — AVANT make dev
export DOCKER_DEFAULT_PLATFORM=linux/amd64
export PYTHONIOENCODING=utf-8 PYTHONUTF8=1

make dev
----

Accès : Frontend http://localhost:8080 · API http://localhost:9000/docs

Détails : xref:04-operationnel/installation/local-minikube.adoc[Démarrage local Minikube].

== Conventions clefs

* *Backend* : FastAPI + Pydantic `BaseSettings`, SQLAlchemy + Alembic (une `version_table` par service sur la base `ibis_x_db` partagée).
* *Tâches async* : Celery avec 3 queues — `ml_queue`, `ai_queue`, `xai_queue`.
* *Frontend* : Angular 19 + Material (pas de Tailwind), i18n FR/EN obligatoire via `@ngx-translate`.
* *Infra* : Kubernetes (Minikube local / AKS Azure) via Kustomize, images préfixées `ibis-x-`.

Voir xref:03-technique/stack-technique.adoc[Stack technique raisonnée] pour les justifications.
