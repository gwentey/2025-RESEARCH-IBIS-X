= Roadmap
:description: Phases passées, en cours, à venir

== Phase 0 — Fondations (réalisée)

* Infrastructure Kubernetes (Minikube + AKS).
* Skaffold + Kustomize.
* Authentification JWT + OAuth.
* Base PostgreSQL partagée avec version_tables.

== Phase 1 — Sélection de datasets (réalisée)

* Modèle multi-fichiers (`storage_path`, `dataset_files`, `file_columns`).
* Catalogue avec filtres.
* Upload + manifest.
* Import Kaggle batch.

== Phase 2 — Pipeline ML (réalisée)

* Wizard 5 étapes frontend.
* Preprocessing configurable.
* Training async Celery (`ml_queue`).
* Métriques + visualisations complètes.
* Renommage `model_uri` → `artifact_uri` (migration 002).

== Phase 3 — Analyse qualité IA (réalisée)

* Intégration OpenAI sur `ai_queue`.
* Table de cache `data_quality_analyses`.
* Invalidation par `dataset_version`.

== Phase 4 — Moteur XAI (réalisée)

* Worker séparé (`Dockerfile.worker`, image `ibis-x-xai-engine-worker`).
* SHAP + LIME + LLM.
* Chat contextuel (5 questions max).
* Optimisation `generate_explanation_with_precalculated_shap`.
* Toutes les tâches sur `xai_queue`.

== Phase 5 — Documentation & conformité (réalisée — 2026-04)
