== 📊 KPIs Obligatoires ✅

=== Métrique Principale
* **MAE (Mean Absolute Error)** : Erreur absolue moyenne, métrique principale pour la régression
** Exprimée dans l'unité de la variable cible
** Plus intuitive que RMSE pour l'interprétation

=== Métriques Secondaires
* **RMSE** : Racine de l'erreur quadratique moyenne
** Pénalise davantage les grandes erreurs
** Utile quand les outliers sont critiques

* **R²** : Coefficient de détermination
** Pourcentage de variance expliquée par le modèle
** Plus proche de 100% = meilleur modèle

* **MSE** : Erreur quadratique moyenne (métrique intermédiaire)

== 🎯 Visualisations Obligatoires Implémentées

=== 1. Scatter Plot : Prédictions vs Valeurs Réelles
* **Objectif** : Évaluer la précision globale du modèle
* **Interprétation** : Les points doivent suivre la diagonale idéale
* **Amélioration** : Affichage des statistiques R² et MAE directement sur le graphique
* **Clé backend** : `regression_plot`

=== 2. 🆕 Résidus vs Prédictions 
* **Objectif** : Détecter l'hétéroscédasticité et les patterns dans les erreurs
* **Interprétation** : Points distribués aléatoirement autour de 0, sans pattern visible
* **Détection** : Variance non-constante, biais systématique
* **Clé backend** : `residual_vs_predicted`

=== 3. 🆕 Histogramme des Résidus
* **Objectif** : Vérifier l'hypothèse de normalité des erreurs
* **Interprétation** : Distribution symétrique, centrée sur 0, approximativement normale
* **Amélioration** : Courbe de distribution normale en overlay pour comparaison
* **Statistiques** : Calcul automatique de l'asymétrie (skewness) avec interprétation
* **Clé backend** : `residuals_histogram`

=== 4. Visualisation de l'Arbre de Décision
* **Objectif** : Comprendre la logique de décision étape par étape
* **Interprétation** : Suivre les branches pour comprendre les prédictions
* **Spécificité** : Arbre simple et interprétable pour régression
* **Clé backend** : `tree_structure`
