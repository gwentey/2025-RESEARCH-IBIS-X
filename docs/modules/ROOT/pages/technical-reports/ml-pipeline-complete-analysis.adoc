.Logique de Recommandation IA
[source,javascript]
----
getAIRecommendation(): 'classification' | 'regression' {
  const columnType = this.getSelectedTargetColumnType();
  const targetColumn = this.getSelectedTargetColumn();
  
  // Analyse du type de données
  if (columnType.includes('object') || 
      columnType.includes('string')) {
    return 'classification';
  }
  
  // Analyse sémantique du nom
  if (targetColumn.includes('class') || 
      targetColumn.includes('type') ||
      targetColumn.includes('label')) {
    return 'classification';
  }
  
  if (targetColumn.includes('price') || 
      targetColumn.includes('amount') ||
      targetColumn.includes('score')) {
    return 'regression';
  }
  
  // Défaut basé sur le type numérique
  return columnType.includes('float') ? 
         'regression' : 'classification';
}
----

=== Étape 4 : Sélection de l'Algorithme

*Objectif :* Choisir l'algorithme ML approprié

==== Algorithmes Disponibles

.Decision Tree 🌲
* *Avantages :*
  - Interprétabilité maximale
  - Pas besoin de normalisation
  - Gère les non-linéarités
* *Inconvénients :*
  - Tendance au surapprentissage
  - Instabilité

.Random Forest 🌳
* *Avantages :*
  - Meilleure généralisation
  - Feature importance native
  - OOB Score gratuit
* *Inconvénients :*
  - Moins interprétable
  - Plus gourmand en ressources

==== Assistance IA pour le Choix

L'IA recommande l'algorithme optimal selon :

- Taille du dataset (<1000 lignes → Decision Tree)
- Besoin d'interprétabilité (critique → Decision Tree)
- Performance maximale (priorité → Random Forest)
- Données bruitées (présentes → Random Forest)

== Processus d'Entraînement Backend

=== Workflow Celery

[source,python]
----
@celery_app.task(bind=True, name='train_model')
def train_model(self, experiment_id: str):
    """
    Workflow d'entraînement complet
    """
    # 1. Validation (10%)
    experiment = validate_experiment(experiment_id)
    
    # 2. Chargement (30%)
    dataset = load_dataset(experiment.dataset_id)
    
    # 3. Preprocessing (50%)
    X_train, X_test, y_train, y_test = preprocess_data(
        dataset,
        experiment.preprocessing_config
    )
    
    # 4. Entraînement (70%)
    model = train_algorithm(
        experiment.algorithm,
        experiment.hyperparameters,
        X_train, y_train
    )
    
    # 5. Évaluation (90%)
    metrics = evaluate_model(
        model, X_test, y_test,
        task_type=experiment.task_type,
        algorithm_type=experiment.algorithm
    )
    
    # 6. Sauvegarde (100%)
    save_results(experiment_id, model, metrics)
----

=== Cache Intelligent

[source,python]

class DataQualityCache:
    TTL = 7 * 24 * 60 * 60  # 7 jours
    
    def get_or_compute(self, dataset_id: str):
        cached = self.get_from_cache(dataset_id)
        if cached and not self.is_expired(cached):
            return cached
        
        analysis = self.compute_analysis(dataset_id)
        self.save_to_cache(dataset_id, analysis)
        return analysis
