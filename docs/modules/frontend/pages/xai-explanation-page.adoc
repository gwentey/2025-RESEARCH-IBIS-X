=== Méthodes XAI : LIME vs SHAP

==== Interface à Onglets

Trois onglets interactifs permettent d'explorer :

1. **LIME** (🔍) : Explications locales
2. **SHAP** (⚖️) : Explications basées sur la théorie des jeux
3. **Comparaison** (🔄) : Analyse comparative

==== Contenu LIME

* **Nom complet** : Local Interpretable Model-agnostic Explanations
* **Description** : Technique d'explication locale pour décisions individuelles
* **Fonctionnement** : Perturbation des données autour d'une instance
* **Points forts** :
  - Explications locales précises
  - Compatible avec tout algorithme
  - Interprétation humaine naturelle
  - Détection de biais locaux

* **Cas d'usage** :
  - Décisions médicales critiques
  - Approbation de crédit
  - Détection de fraude
  - Classification d'images

* **Exemple visuel** : Analyse d'approbation de prêt avec barres d'importance

==== Contenu SHAP

* **Nom complet** : SHapley Additive exPlanations
* **Description** : Méthode basée sur la théorie des jeux de Shapley
* **Fonctionnement** : Calcul de contributions équitables des caractéristiques
* **Points forts** :
  - Contributions mathématiquement justifiées
  - Explications globales et locales
  - Propriétés théoriques garanties
  - Visualisations avancées

* **Cas d'usage** :
  - Analyse de performance produit
  - Optimisation marketing
  - Recherche scientifique
  - Modèles de pricing

* **Exemple visuel** : Graphique en cascade (waterfall) pour prix immobilier

==== Comparaison LIME vs SHAP

Tableau comparatif présentant :

* **Portée** : Local vs Global/Local
* **Approche** : Perturbation vs Théorie des jeux
* **Utilisation optimale** : Décisions individuelles vs Analyse globale

===== Guide d'Utilisation

Scénarios recommandés pour chaque méthode :

**Utilisez LIME quand :**
- Explication d'une décision spécifique requise
- Impact direct sur un individu
- Besoin d'explication simple pour non-experts

**Utilisez SHAP quand :**
- Analyse de performance globale du modèle
- Optimisation des caractéristiques importantes
- Recherche ou analyse scientifique
