= Guide IA pour Datasets : Assistant Intelligent ML
:navtitle: Guide IA Datasets
:description: Documentation du nouveau composant Guide IA qui remplace l'onglet Analytics pour offrir des recommandations ML personnalisées

== Vue d'ensemble

Le **Guide IA** est un nouveau composant intelligent qui remplace l'ancien onglet "Analytics" dans la page de détail des datasets. Il utilise l'intelligence artificielle pour analyser automatiquement un dataset et fournir des recommandations personnalisées sur la meilleure approche ML à adopter.

== Fonctionnalités Principales

=== Analyse Automatique du Dataset
L'IA analyse automatiquement :

* **Métadonnées** : Nom, objectif, domaine, tâches ML
* **Structure** : Nombre d'instances, features, fichiers
* **Qualité** : Pourcentage de données manquantes, anonymisation
* **Contexte** : Année, disponibilité, domaine d'application

=== Recommandations ML Intelligentes

==== Tâche Recommandée
L'IA détermine automatiquement entre :

* **Classification** : Pour prédire des catégories ou classes distinctes
* **Régression** : Pour prédire des valeurs numériques continues

==== Algorithme Recommandé
Choix intelligent entre :

* **Decision Tree** : Pour datasets modérés, résultats interprétables
* **Random Forest** : Pour datasets volumineux, meilleure robustesse

=== Interface Utilisateur Moderne

==== Hero Section
* Icône gradient moderne avec effet d'ombre
* Call-to-action claire "Analyser avec l'IA"
* Description explicative du processus

==== Processus d'Analyse Animé
Étapes visuelles avec indicateurs de progression :

1. **Lecture des métadonnées** 📖
2. **Analyse des patterns** 📊  
3. **Génération des recommandations** 💡

==== Résultats Structurés
* **Analyse du Dataset** : Résumé de ce que l'IA comprend
* **Recommandations ML** : Tâche et algorithme avec justifications
* **Points Clés** : Insights essentiels à retenir
* **Points d'Attention** : Défis potentiels à anticiper

== Logique d'Analyse Intelligente

=== Détermination de la Tâche ML

L'algorithme suit cette logique :

[source,typescript]
----
if (dataset.task.includes('classification')) {
  return 'classification';
} else if (dataset.task.includes('regression')) {
  return 'regression';  
} else if (isEducationDomain) {
  return 'classification'; // Domaine éducatif = souvent classification
} else {
  return 'classification'; // Par défaut
}
----

=== Sélection de l'Algorithme

[source,typescript]
----
if (instances > 10000 && features > 20) {
  return 'random_forest'; // Volume important
} else {
  return 'decision_tree'; // Dataset modéré
}
----

== Avantages par rapport à Analytics

=== Avant (Analytics)
* ❌ Tables statiques de corrélations
* ❌ Patterns complexes difficiles à comprendre  
* ❌ Aucune recommandation actionnable
* ❌ Interface technique peu accessible

=== Après (Guide IA)
* ✅ **Analyse intelligente** avec recommandations
* ✅ **Interface conversationnelle** moderne
* ✅ **Conseils actionnables** pour démarrer le ML
* ✅ **Expérience guidée** pour utilisateurs non-experts
