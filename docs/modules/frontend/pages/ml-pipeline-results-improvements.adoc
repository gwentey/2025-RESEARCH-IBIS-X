= Améliorations Page Résultats ML Pipeline - Focus sur XAI
:description: Documentation technique des améliorations apportées à la page des résultats d'expérience ML pipeline avec mise en valeur de l'explicabilité XAI
:keywords: ml-pipeline, results, xai, explicabilité, shap, lime, decision-tree, random-forest
:page-layout: doc

Cette page documente les améliorations majeures apportées à la page des résultats d'expérience ML pipeline (`experiment-results.component`) pour résoudre le problème d'affichage de l'importance des variables et mettre en valeur l'explicabilité XAI.

== Problème Identifié

=== Situation Initiale
La page des résultats d'expérience ML pipeline affichait une *feature importance basique* de scikit-learn dans la section "Visualisations du Modèle", qui :

* Utilisait simplement `model.feature_importances_` de scikit-learn
* N'était pas basée sur SHAP ou LIME
* Était peu informative pour comprendre les prédictions
* Coexistait avec une vraie section XAI qui utilisait SHAP/LIME

=== Validation Technique
Après vérification du service XAI (`xai-engine-service/app/xai/explainers.py`), nous avons confirmé que :

* ✅ Le service XAI utilise bien **SHAP TreeExplainer** pour Decision Tree et Random Forest
* ✅ Le service XAI utilise **LIME** pour les autres modèles  
* ✅ La fonction `choose_best_explainer()` sélectionne automatiquement la méthode appropriée
* ❌ La "feature importance" dans les visualisations n'était pas SHAP/LIME

=== 2. Amélioration de la Section XAI

==== Interface Améliorée
* **Titre enrichi** : "Explicabilité du Modèle (XAI)" avec icône
* **Badges informatifs** : Méthode recommandée, algorithme, type de tâche
* **Card d'introduction** : Explication des bénéfices de XAI
* **ID de section** : Pour navigation par scroll automatique

==== Méthodes TypeScript Ajoutées
[source,typescript]
----
getRecommendedXAIMethod(): string {
  const algorithm = this.getAlgorithm();
  if (algorithm === 'decision_tree' || algorithm === 'random_forest') {
    return 'SHAP TreeExplainer';
  }
  return 'LIME';
}

scrollToXAISection(): void {
  const element = document.getElementById('xai-explanation-section');
  if (element) {
    element.scrollIntoView({ 
      behavior: 'smooth', 
      block: 'start' 
    });
  }
}
----

getMetricDescription(key: string): string {
  const algorithm = this.getAlgorithm();

    }`,
    'precision': `Précision des prédictions positives ${
      algorithm === 'random_forest' ? '(moyenne des arbres)' : ''
    }`,
    // ... autres métriques
  };
  return descriptions[key] || 'Métrique inconnue';
}

== Différenciation Classification vs Régression

=== Affichage Adapté
* **Classification** : Matrice de confusion, courbe ROC, métriques accuracy/precision/recall/f1
* **Régression** : Scatter plot prédictions vs réalité, métriques R²/MAE/MSE/RMSE
* **Ordre des métriques** : R² en premier pour la régression (plus importante)

=== Recommandations Contextuelles
* **Classification** : Analyse des erreurs de classification via XAI
* **Régression** : Analyse des prédictions extrêmes via XAI

== Différenciation Decision Tree vs Random Forest

=== Messages Adaptatifs
[source,typescript]
----
getDetailedAnalysis(): string {
  const algorithm = this.getAlgorithm();
  
  if (algorithm === 'random_forest') {
    if (score >= 85) {
      return `Random Forest offre une excellente robustesse grâce à 
              l'agrégation de ${this.getTreeCount()} arbres. 
              Utilisez XAI pour comprendre comment ces arbres collaborent.`;
    }
  } else if (algorithm === 'decision_tree') {
    if (score >= 85) {
      return `Excellent résultat pour un arbre simple ! 
              L'explicabilité XAI vous permettra d'explorer 
              chaque branche de décision en détail.`;
    }
  }
}
----

=== Recommandations Algorithmiques
* **Decision Tree < 85%** → Recommander Random Forest + XAI pour comparaison
* **Random Forest ≥ 90%** → Comparer avec Decision Tree + XAI pour évaluer la complexité
